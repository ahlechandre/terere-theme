#Terere Theme
##### theme for Adobe Brackets

## Install
> 1. ```Extension Manager > Themes > Terere ```

> or

> 1. ``` git clone https://github.com/ahlechandre/terere-theme.git /someFolder```

> 2. ``` Help > Show Extensions Folder > User```

> 3. paste this project

and 

> ```View > Theme > Terere```
